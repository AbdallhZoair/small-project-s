This is where small ideas come to live instead of dying alone in 15 separate repos with names like `test-final-v2`.
Think of it as a sandbox — not everything here is meant to impress, some of it is just me messing around with an idea for an evening. But it's organized, it runs, and nothing here is a mess pretending not to be one.
---
🗂️ How this is organized
Every idea gets its own folder.
Every folder has one job and does it.
Naming follows `snake_case` — consistency is free, so why not.
That's the whole system. Simple enough that I don't need to explain it again next time I add something.
---
▶️ Running any of these
```bash
cd <folder-name>
python3 <script>.py
```
No installs, no config files, no "works on my machine" surprises.
---
💡 The philosophy here
Not every project needs its own spotlight. Some ideas are quick sketches, not paintings — and that's fine, as long as they live somewhere organized instead of cluttering a profile with a graveyard of one-commit repos.
The bigger stuff, the things I actually put real thought into, get their own repo with a real README and an actual plan. This folder is just where I keep sharp. 🔪
